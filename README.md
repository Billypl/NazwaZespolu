# An online shop
Online shop made with PrestaShop for Electronic Buisness class.

# Used software
- PrestaShop 1.7.8.11
- MariaDB 11.5.2
- Docker

# How to run locally (dev)?
To run the shop:
```
./run.sh # foreground
./run.sh -d # background
```
To stop and cleanup docker:
```
./stop.sh
```
